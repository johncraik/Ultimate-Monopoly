## 7. 🌟 Card action services — built in increments
All as `CardAction` subclass + `ICardActionService<T>` + `CardService` switch arm + DI line (the
established open/closed pattern). Built this session:
- **Turns** (`TurnsActionService` — miss/extra), **Direction** (`DirectionActionService` — flip,
  any target), **Loans** (`LoansActionService` — wipe-all / repay-all), **Building**
  (`BuildingActionService` — purge self/chosen, + **GrantHotel**), **Property**
  (`PropertyActionService` — return-to-bank / hand-in-FP, with a **`Set` bool**, plus
  **TakeFromBank / ReceiveAllFreeParking / ClearFreeParkingToBank**).
- **`PlayerTarget.Everyone`** (self + all others) added + wired into `CardActionHelper.ResolveTargets`.
- **Movement**: `NearestOwnedByOther` (nearest-of-kind owned by another) + **`MovementJailFilter`**
  (`OnlyJailed`/`OnlyNotJailed` — mass breakout / call-a-meeting).
- **Jail**: `JailKind.ModifyLeaveFee` (set £50 / multiply) on `JailCost`.
- **GlobalEvent** (`GlobalEventActionService` → `GlobalEventService.Start*Event`) — the 5 Double
  global-event cards. **DeckDraw** (`DeckDrawActionService` — recurses `engine.CardService.DrawCard`,
  no DI cycle) — the "or take a Chance" cards.
- **Money** extensions: `Target` (each-player grants *from the bank*, realised per subject), an
  amount **`Basis`** (`PercentOfOwnCash` / `PercentOfFreeParkingPot` / `TripleBonus` /
  `SnakeEyesBonus`), and **`SwapCash`** (swap balances with a chosen / dice-off player). Reuses
  existing fields — no new model state.
- **`LoansAction.WipeAllAndRewardClear`** — the deterministic compound: snapshot loan-free players
  *before* the wipe → wipe everyone → £1000 + forced property-return to the snapshot-clear players
  (reuses `PropertyActionService` for the return; £1000 kept even with nothing to give back).
